# DirkOdds Android Executive Summary

DirkOdds Mobile is an Android prototype for testing prediction interactions through abstract live-action sports simulation rather than text-only outputs.

This package currently provides:

- a fictional football, basketball, and baseball scenario deck
- an interactive pre-match prediction lane selector
- a real-time 3D OpenGL ES playback view
- anonymous, low-detail standee actors and synthetic venues
- live HUD bars for prediction edge, momentum, control, reflex, and pressure
- lightweight coaching interactions: `Pulse Pick`, `Set Shape`, and `Reflex Tap`

What this build is suitable for:

- user interaction testing
- prediction workflow review
- timing and QTE feel review
- client demonstration of a non-text prediction experience

What this build is not yet:

- not a licensed production sports title
- not a regulated wagering product
- not a live real-league data client
- not a final identity-cleared release

The design intent is to let stakeholders evaluate the interactive DirkOdds concept without exposing real sports identities or sensitive source data.
