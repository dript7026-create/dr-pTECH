- Style: "8-bit Game Boy pixel art, crisp pixels, hard edges, no blur"
- Lighting: "flat, high-contrast silhouette"
- Composition: centered on transparent background, pivot at bottom-center unless otherwise specified

Final generation notes for all prompts
- Request "no anti-aliasing" and "nearest-neighbor scaling" (if engine supports), so pixels are clean.
- For animated sprite-sheets, request exact output canvas equal to (frame_count * frame_width) x frame_height.
- If Recraft cannot directly output indexed 2bpp, request a high-contrast PNG and then run `convert_assets.py` to reduce to 4-shade indexed PNG.
- When requesting variants (color swaps), ask for layers or multiple passes with the same line-art to preserve silhouette.
