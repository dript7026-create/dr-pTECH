# TOMMY GOOMBA
An original story by the Carell brothers for the Nintendo Game Boy family of systems.

## Overview
"TOMMY GOOMBA" is an action-adventure game set in the Mushroom Kingdom. Players control Tommy, a father on a quest to rescue his wife after a chaotic encounter with two mysterious figures. The game features a variety of stages, combat mechanics, and a unique save system.

## Story
The game begins in Tommy's house, where he lives with his wife and children. Suddenly, two strange figures crash through the ceiling, causing chaos and leading to the tragic loss of Tommy's children. The figures kidnap Tommy's wife and escape, leaving behind only laughter from their cousins on Mario's new iPhone. 

As the house floods with tears, Tommy is shocked awake by the iPhone shorting out in his tears, granting him electrically charged speed abilities. He must navigate through 66 prebuilt stages to find his wife and confront the brothers.

## Gameplay
- **Objective**: Find Tommy's wife while navigating through various stages.
- **Enemies**: Encounter the two brothers and their cousins throughout the game.
- **Combat**: Engage in battles with the brothers and defend against the cousins.
- **Endings**: Achieve different endings based on the player's choices and encounters.

## Project Structure
- **src/**: Contains the main game logic and functionality.
- **include/**: Header files for declarations and structures.
- **assets/**: Graphics and map data for the game.
- **Makefile**: Build instructions for compiling the project.
- **gbdk.mk**: Configuration for the Game Boy Development Kit.

## Installation
1. Clone the repository.
2. Navigate to the project directory.
3. Run `make` to compile the project.
4. Load the generated `.gb` file onto your Game Boy or emulator.
