1. Use the repo-local installer path: `c:\Users\rrcar\Documents\drIpTECH\.tools\quake2-sdk\Quake-2`.
2. In PowerShell run: `.\.tools\bin\q2sdk-env.ps1`.
3. Build with: `.\.tools\bin\bango-idtech2-build.ps1`.
4. The build compiles against real Quake II SDK headers from `%Q2_SDK_DIR%\game` and emits a `game` shared module (`gamex86.dll` on Windows).
5. To bind direct `gi.*` runtime imports in a full engine patch, compile with `-DBANGO_Q2_RUNTIME_LINKED=1` inside the host engine build.
6. Copy the built module into your selected Quake II mod folder and stage assets from `generated/idLoadINT_bundle/`.
