# DoENGINE

This is the parent project for DoENGINE, a future-proof, frequency-based, motion/vibration-driven engine and framework. All subprojects, modules, and integrations (including playhub and metroidvania-hacknslash) are to be reintegrated here.

## Structure
- /src: Core engine code
- /metadata: Permanent, encrypted traces and metadata
- /encryption: Proprietary encryption modules
- /backup: Threefold backup system
- /philosophy: Documentation of nomenclature, philosophy, and hardware trace principles

## Integration
- All previous and future projects are to be integrated as DoENGINE modules or plugins.
- Playhub philosophy and modules are fully compatible and traced.
